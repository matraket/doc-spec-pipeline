#### US-167: Dashboard adaptado por rol

**Prioridad:** Should

> Como **sistema**,

> para que cada usuario vea la información relevante.

```gherkin
Scenario: Dashboard de tesorero

    | Próximas remesas     |
    | Morosos destacados   |

  Given un usuario con rol Secretario

    | widget prioritario     |

    | widget                 |
    | Mi estado de cuotas    |
    | Próximos eventos       |
    | Últimos avisos         |
```
